Profile Template

Profile 模板

The AgentProfile schema — a standardized format for AI agents.

AgentProfile 规范 — AI Agent 的标准化配置格式。

agent-profile.json 
{
  // Required fields
  "name":           "Python Code Review Assistant",
  "description":    "An AI assistant for Python code review...",
  "details":        "## Role\n\n...\n\n## Core Dimensions\n\n...",

  // Optional fields
  "agent_template": "default",
  "tools":          [],
  "skills":         ["python-lint", "security-scan"],
  "subagents":      []
}

Details Format

Details 格式

The details field uses Markdown with four sections that define the agent's behavior.

details 字段使用 Markdown 编写，包含四个部分来定义 Agent 的行为。

Section 1

Role

Define the agent's identity, tone, and behavioral philosophy. Specify the domain, function, and core approach.

Section 2

Core Dimensions

Structured breakdown of focus areas using tables or lists. Each dimension includes specific focus points.

Section 3

Standards

Quality criteria, industry standards, evaluation methods, and feedback methodology.

Section 4

Output Format

Define the structured response format: how the agent should organize and present its output.

第一部分

角色定位

定义 Agent 的身份、语气和行为理念。指定领域、功能和核心方法。

第二部分

核心维度

使用表格或列表对关注点进行结构化分解，每个维度包含具体的关注要点。

第三部分

标准规范

质量标准、行业规范、评价方法和反馈方式。

第四部分

输出格式

定义结构化的响应格式：Agent 应如何组织和呈现其输出内容。
